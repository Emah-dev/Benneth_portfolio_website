Web Design + Development

Web design and development is an umbrella term that describes the process of creating a website, it involes two major skills sets: web design and web development. Web design is the look and feel of the website, while web development determines how it functions.

Computer Operating

A computer operation is an action carried out to accomplish a given task in a computer system. A computer operator oversees the running of the computer systems, ensuring that the machines and computers are running properly.

Hardware & Software Installation

The installation of hardware is the process of installing physical components into a computer or an electronic device, while software software installation is the process of putting data onto some kind of storage so that it can be accessed by other devices.
